✕ Without SEO (anti-pattern)

  • Generic title: Angular App
  • No meta description
  • No robots rule
  • No canonical link
  • No Open Graph / Twitter tags
  • No JSON-LD structured data

Social shares and Google snippets will look poor and untrustworthy.

✓ With SEO (recommended)

  • Unique, descriptive <title>
  • Meta description for snippets
  • meta name="robots"
  • link rel="canonical"
  • Open Graph: title, description, image, url
  • Twitter Card tags
  • JSON-LD WebPage schema

Use SeoService.apply() on every public route.
