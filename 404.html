URL & route SEO

Clean routes are an SEO signal — even on the homepage

✓ What we do
  • https://site.com/
  • https://site.com/products
  • https://site.com/blog/understanding-ssr
✕ What we avoid
  • https://site.com/#/home
  • https://site.com/index.html?page=home
  • https://site.com/?p=1&ref=nav&utm_x=1

💡 Use real, crawlable paths (not hash #/ routes), lowercase words, and hyphens. Each public page should have ONE canonical URL — extra query params create duplicate URLs that dilute ranking.
